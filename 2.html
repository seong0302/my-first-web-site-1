CSS

A Cascading Style Sheet ( English : Cascading Style Sheet ) is a style language ( English : Style sheet language [ * ] ) that describes how the markup language is actually displayed [1] . It is mainly used for XHTML , but can also be used for XML . It is a W3C standard, and it has a high degree of freedom when defining layout and style .
